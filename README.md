# Task Manager Backend

## Setup
1. Clone the repo
2. `npm install` 
3. Create `.env` from `.env.example` and add MongoDB connection string
4. `npm run dev` (local) or `npm start` (production)

## API Endpoints
- `GET /api/users/:id/dashboard` — Get user's dashboard
- `POST /api/tasks` — Create a task
- `PUT /api/tasks/:id/complete` — Mark task complete
